# 💻 Estrutura de Dados - Conceitos e Represetação

Este repositório foi criado para documentar minha evolução na disciplina de **Estrutura de Dados**. Aqui, submeto os desafios práticos propostos em aula, organizados por níveis de complexidade.

O foco dos projetos é o domínio da linguagem **C**, abrangendo conceitos como alocação dinâmica de memória, structs, ponteiros e lógica de algoritmos.

## 🚀 Progresso dos Desafios

### 🟢 Nível Novato (Fundamentos)
* **Biblioteca de Livros (Parte 1):** Sistema básico de cadastro utilizando estruturas simples e manipulação de strings.
* **War (Preparando o Terreno):** Início da lógica de territórios, focando em entrada e saída de dados e estruturação de dados.

### 🟡 Nível Aventureiro (Intermediário)
* **Biblioteca de Livros (Parte 2):** Evolução para o uso de **Alocação Dinâmica (`malloc`/`calloc`)** e sistemas de empréstimos vinculados.
* **War (Vamos ao Ataque):** Implementação de **Aritmética de Ponteiros**, funções modulares, geradores de números aleatórios e lógica de combate/conquista de territórios.
